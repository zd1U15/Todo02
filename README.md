## 主な機能
* **在庫の一覧表示**: 登録されている在庫商品を一覧表示できます。
* **在庫追加・編集・登録・削除**: 在庫商品の追加・編集・削除ができます。
* **受注情報一覧表示**: 現在までの注文履歴を一覧で表示します。
* **注文履歴表示**: 登録されている顧客の注文履歴と顧客データを表示します。


## 技術スタック

このアプリケーションは、以下の技術とフレームワークを使用して構築されています。

| カテゴリ | 技術名 | 説明 |
| :--- | :--- | :--- |
| **バックエンド** | Java 21 | アプリケーションの核となるプログラミング言語です。 |
| **Webフレームワーク** | Spring Boot 3 | JavaでWebアプリを効率よく作るための土台です。 |
| **テンプレートエンジン** | Thymeleaf | HTMLにデータ（タスク一覧など）を埋め込むために使っています。 |
| **データベース** | MySQL | タスクデータを保存・管理するためのデータベースです。 |
| **フロントエンド** | HTML5 / CSS3 / JavaScript | 画面のデザインと構造を担当しています。 |
