## 🎯 概要と目的

このプロジェクトは、Webアプリケーション開発の基礎を学ぶために作成されたシンプルなToDoリスト管理アプリです。

**特に以下のような学習者を対象としています:**

* Spring BootとThymeleafの基本的な連携方法を知りたい方
* 簡単なCRUD操作（作成・読み取り・更新・削除）の実装例を見たい方

## ✅ 主な機能

* **タスクの登録**: 新しいToDo項目をリストに追加できます。
* **タスクの表示**: 登録されている全てのタスクを一覧で表示します。
* **タスクの完了マーク**: 完了したタスクにチェックをつけ、状態を管理できます。
* **タスクの削除**: 不要になったタスクをデータベースから削除します。

## 💻 使用技術 (Technology Stack)

このアプリケーションは、以下の技術とフレームワークを使用して構築されています。

| カテゴリ | 技術名 | 説明 |
| :--- | :--- | :--- |
| **バックエンド** | Java 21 (またはあなたが使っているバージョン) | アプリケーションの核となるプログラミング言語です。 |
| **Webフレームワーク** | Spring Boot 3 | JavaでWebアプリを効率よく作るための土台です。 |
| **テンプレートエンジン** | Thymeleaf | HTMLにデータ（タスク一覧など）を埋め込むために使っています。 |
| **データベース** | H2 Database (または MySQL など) | タスクデータを保存・管理するためのデータベースです。 |
| **フロントエンド** | HTML / CSS | 画面のデザインと構造を担当しています。 |
